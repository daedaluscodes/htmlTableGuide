HTML Table Reference

Tag Name Description
<table> table content used to represent a two-dimensional table made of rows and columns
<tbody> table body contains all data except header and footer
<thead> table head defines the heading of columns in table rows
<tfoot> table footer summarizes content at end of table
<th> table heading enclosed in table rows; used to add titles to rows and columns
<tr> table row adds row to table before adding table data and headings
<td> table data nested inside of a table row to add a cell of data to the table

Attribute Name Description
colspan=" "; column span used on <th> or <td> to span columns; value between 1 and 1000
rowspan=" "; row span used on <th> or <td> to span rows; value between 1 and 1000
